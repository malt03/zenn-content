# リポジトリのクローン

https://github.com/malt03/othello-book

本書内のコードは全てこのリポジトリに含まれています。

フォークしてクローンすることをお勧めします。  
面倒な方は、そのままクローンしても構いません。

```
git clone https://github.com/malt03/othello-book.git
```
